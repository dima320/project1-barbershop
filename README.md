# [Назва барбершопу] — сайт

## Структура проєкту

```
index.html                  — головна сторінка (one-page, якорні секції)
assets/
  css/
    reset.css                — базовий CSS reset
    variables.css             — токени: кольори, шрифти, відступи
    layout.css                — розкладка секцій, сітки, header/footer
    components.css            — кнопки, картки, типографіка, дрібні елементи
    style.css                 — головний файл, підключає всі інші (підключати в HTML лише його)
  js/
    config.js                 — ЄДИНЕ місце для контактів/посилань (запис, телефон, адреса, соцмережі)
    navigation.js              — sticky-header, мобільне меню, скрол-прогрес, reveal-анімації
    counters.js                — анімація чисел у блоці статистики
    slider.js                  — карусель відгуків
    main.js                    — точка входу, викликає ініціалізацію модулів
  images/
    hero/ services/ team/ gallery/ icons/  — папки під фото (зараз — плейсхолдери)
robots.txt
sitemap.xml
```

## Що потрібно замінити перед публікацією

Усі місця, що потребують реальних даних, позначені `[У ДУЖКАХ]` або коментарем `TODO` у коді:

1. **`assets/js/config.js`** — посилання на запис (`bookingUrl`), телефон, адреса, координати, соцмережі, Google Maps embed.
2. **`index.html`**:
   - Назва барбершопу (`[НАЗВА БАРБЕРШОПУ]`, `[NAME]`) — по всьому файлу
   - Місто/адреса — по всьому файлу
   - Ціни послуг (`[000]₴`)
   - Імена майстрів (`[Ім'я майстра]`)
   - `<meta>` теги (title/description/og:image) і JSON-LD блок (Schema.org LocalBusiness) — підставити реальні дані компанії
   - `<iframe src="">` у секції контактів — вставити реальний Google Maps embed URL
3. **Фото** — замінити всі `<div class="media-placeholder">...</div>` на `<img src="/assets/images/..." alt="...">` з реальними фото (папки вже підготовлені за розділами: hero, services, team, gallery).
4. **`robots.txt` / `sitemap.xml` / `og:url` / `canonical`** — замінити `example.com` на реальний домен.

## Технічні нотатки

- Шрифти: **Bebas Neue** (заголовки) + **Manrope** (текст), підключені через Google Fonts.
- Кольорова схема: вугільно-чорний фон + латунний акцент (див. `variables.css`, змінні `--color-*`).
- Сайт адаптивний (мобільне меню, сітки перестроюються під малі екрани), враховано `prefers-reduced-motion`.
- SEO: семантичний HTML5, meta/OG/Twitter теги, Schema.org `HairSalon` (GEO SEO — адреса + координати), `robots.txt`, `sitemap.xml`.
- Кнопка «Записатися» по всьому сайту використовує атрибут `data-booking-link` — посилання підставляється автоматично з `config.js`, міняти в одному місці.

## Як переглянути локально

Відкрити `index.html` у браузері, або (рекомендовано, щоб коректно працювали шляхи `/assets/...`) запустити локальний сервер у папці проєкту:

```
npx serve .
```
